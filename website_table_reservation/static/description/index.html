POS Table Reservation From Website

Main Features

  • Website table reservation primarily provide a platform that allows customers to book online table reservations.
  • Provide greater convenience for customers to reserve a table for party, meetings, gathering etc.
  • Easy to book the table as per the calendar date and timings of the shop.
  • Online bookings with real time availability of place.
  • Using this module you can create diffrenet floors and tables, Also arrange them as per your requirement.

Steps for booking

Select the date & time > select table.


Also, You can see he table info by clicking on icon.

Above table view is coming from the POS, Arrange the table positions in POS and that will directly visible on website for information.

When the customer books any table, it will show as booked table status.

After the order confirmation you can see the table reservation reference in the sale order.

Also, You can see all table reservations under the table reservation menu.

v16.0.1.0

FixedFixed a bug where if multiple slots are selected but not in sequence then start date and end date is showing start date of first slot and end date of last slot. Also even if the table is not booked between that range it shows booked.

For example: Table T1 is booked for 18/12/2023 11:00-11:30 , 14:00-14:30, 16:00-16:30 slots. Then start date was showing 18/12/2023 11:00 and end date was showing 18/12/2023 16:30. Now there are slots between start and end date where table T1 is available but it shows booked between start and end date.
